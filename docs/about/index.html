About Pastry Chasers

The Story Behind Pastry Chasers

Pastry Chasers began with a group of friends who love to run and love pastries. As someone who loves running, exercising, designing and coding - I wanted to work on a project that could incorporate all of these and make a website for a run club where running is only a means of getting to eat more pastries!

While working on Pastry Chasers, I ran into a couple of challenges, particularly with integrating various APIs and ensuring a smooth user experience. However, these obstacles only fueled my passion for the project and pushed me to learn more. While I had initially hoped to create a page with specific running routes and training runs, I couldn't find a suitable API for this.

The Website Journey

This project has been an incredible learning journey. From crafting the user interface to building interactive features, I’ve been able to explore and experiment with modern front-end frameworks and technologies.

Highlights

  • Built with Astro, Svelte, HTML, CSS, Bootstrap, and JavaScript
  • Designed and implemented a responsive layout for seamless use across all devices
  • Integrated external APIs to deliver exercises and form feedback
  • Applied UI/UX best practices to improve accessibility and overall user experience
  • Found some new exercises to test in the gym!
